# Task Manager

This is a simple Task Manager web application created using HTML, CSS, and JavaScript as part of my internship at Algonive.

Users can add tasks with a title, description, and due date. Tasks can be marked as completed or pending, and all data is stored using browser localStorage.

---

### Technologies Used
- HTML
- CSS
- JavaScript

---

### How to Run
Open the `index.html` file in a web browser.
